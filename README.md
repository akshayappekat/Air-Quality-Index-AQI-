# 🌫️ Air Quality Index (AQI) Analysis using K-Means Clustering

## 🚀 Project Overview
This project analyzes air pollution data using a custom implementation of the K-Means clustering algorithm. It groups environmental data into pollution levels (Low, Moderate, High) based on key pollutants like PM2.5, PM10, NO₂, SO₂, and CO.

The project focuses on applying Data Structures and Algorithms (ADS) concepts to real-world datasets for pattern recognition and analysis.

---

## 🎯 Features
- Data cleaning and preprocessing  
- Z-score normalization  
- Custom K-Means clustering implementation  
- Elbow method for optimal cluster selection  
- PCA for dimensionality reduction  
- Correlation heatmap visualization
